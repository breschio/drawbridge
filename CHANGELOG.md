# Changelog

All notable changes to Drawbridge will be documented in this file.

## [1.1.0] - 2026-01-02

### Added
- **AI integration improvements** — Enhanced prompt formatting for Cursor and Claude Code
- **Visual feedback features** — Improved element highlighting and selection
- **Better Cursor compatibility** — Optimized markdown output for Cursor's AI workflows
- **Claude Code support** — Formatted prompts work seamlessly with Claude Code

### Improved
- Screenshot capture quality and performance
- Element selector accuracy
- UI/UX refinements for annotation workflow

### Fixed
- Element detection on complex page layouts
- Screenshot cropping with padding adjustments

## [1.0.0] - Initial Release

### Features
- Point-and-click element selection
- Automatic screenshot capture with context
- CSS selector detection
- Markdown-formatted output for AI assistants
- Local storage for preferences and history
- Works on any website (localhost, staging, production)
